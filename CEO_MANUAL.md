# CEO 운영 매뉴얼 (비기술자용)

이 문서는 **비개발자도 운영/성장 관점에서** 오늘약국을 관리할 수 있도록 정리한 매뉴얼입니다.

## 1) 오늘약국의 핵심 목표

- **사용자 가치**: “지금 문 연 약국”을 빠르게 찾고, 영업시간·전화·길찾기를 한 번에 확인
- **비즈니스 가치**: 검색 노출(SEO/AEO) 기반 트래픽을 안정적으로 확보하고, 애드센스 수익으로 확장

## 2) 구글 노출(SEO) 관점에서 꼭 챙길 페이지

- **홈(`/`)**: 핵심 기능 + 검색 트래픽 유입의 시작점
- **약국 상세(`/pharmacy/[hpid]`)**: 데이터 기반 페이지, 지속 트래픽의 중심
- **소개(`/about`)**: 사이트 신뢰(Who/Why) 강화 → 애드센스/검색 품질에 도움
- **문의(`/contact`)**, **개인정보(`/privacy`)**: 정책/신뢰 필수

## 3) 소개(/about) 페이지를 풍부하게 만든 이유

- 검색엔진은 “이 사이트가 누구를 위해 무엇을 제공하는지”를 중요하게 봅니다.
- `/about`는 **브랜드 신뢰 + 콘텐츠 품질 신호** 역할을 합니다.
- 구성 원칙:
  - 질문형 H1 + 2~4문장 핵심 요약(AEO)
  - 데이터 출처/업데이트 안내(정책 안전)
  - 이용 팁 + FAQ(체류시간/만족도 향상)
  - 내부 링크로 핵심 페이지 연결(SEO 내부링크)

## 4) 운영 체크리스트(주 1회 권장)

- GitHub Actions가 정상 동작하는지 확인 (Generate / Publish / Sync)
- Supabase 데이터가 최신인지 확인 (약국/콘텐츠 큐)
- Search Console에서 색인/오류 점검
